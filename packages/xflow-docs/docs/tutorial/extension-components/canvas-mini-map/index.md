# MiniMap 小地图

提供用于预览全局的小地图，方便用户快速定位自己感兴趣的可视区域

# 何时使用

- 需要预览全局节点时
- 需要通过地图
- 通过地图移动可视区域
- 通过地图放大缩小可视区域
